# Caprica Server

> A server for Caprica Systems

# Usage

```sh
$ yarn start
# or
$ npm start
```

# Pre-requisites

Ask [@mathdroid](https://github.com/mathdroid) for `capricaServiceAccount.json`. It will be required for Firebase connection.

```sh
$ yarn
# or
$ npm install
```
